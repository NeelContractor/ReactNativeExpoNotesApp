# Zustand + MMKV Notes (React Native Expo)

A simple Notes app built with Zustand for state management and react-native-mmkv for ultra-fast persistent storage in a React Native app using Expo.

## This project is built for learning how to combine:

- Global state (Zustand)
- Persistent storage (MMKV)
- Production-ready architecture in Expo

## Features

- Add notes
- Delete notes
- List all notes
- Persistent storage (survives app restart)
- Clean Zustand store architecture
- MMKV high-performance storage
